# SOC Analyst Home Lab

This project is a hands-on security operations center (SOC) lab designed to simulate a small enterprise environment and practice security monitoring, threat detection, and incident investigation.

## Overview

The lab consists of a Windows Server Active Directory environment monitored by Wazuh SIEM and Sysmon. Security events are generated through controlled simulations and analyzed using endpoint telemetry, Windows event logs, and MITRE ATT&CK techniques.

## Objectives

The goals of this project are to gain practical experience with:

- Security event monitoring
- SIEM alert investigation
- Windows event analysis
- Endpoint detection and response (EDR) concepts
- Active Directory security
- MITRE ATT&CK technique mapping
- Incident documentation and response workflows
